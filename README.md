# GitHub User Finder

**Автор:** Тихон Мельянков

## Описание

приложение с графическим интерфейсом для поиска пользователей GitHub

### Основные функции:
- Поиск пользователей по логину (или части логина)
- Отображение списка найденных пользователей
- Двойной клик по пользователю для просмотра детальной информации (имя, компания, локация, количество репозиториев, подписчиков и т.д.)
- Добавление пользователей в избранное
- Просмотр избранных пользователей с возможностью удаления
- Сохранение избранных в файл `favorites.json`
- Валидация ввода (поле поиска не может быть пустым)

## Как использовать API

Приложение использует **GitHub REST API v3** без аутентификации.

### Используемые эндпоинты:

1. **Поиск пользователей**
   - URL: `https://api.github.com/search/users?q={query}&per_page=20`
   - Метод: `GET`
   - Параметры:
     - `q` — поисковый запрос (логин пользователя)
     - `per_page` — количество результатов на страницу (макс. 100)

2. **Детальная информация о пользователе**
   - URL: `https://api.github.com/users/{username}`
   - Метод: `GET`
   - Возвращает: логин, имя, компанию, локацию, email, количество репозиториев, подписчиков, подписок, дату регистрации, URL профиля и краткую биографию.

### Ограничения без аутентификации:
- 60 запросов в час с одного IP-адреса
- Для увеличения лимита можно добавить токен аутентификации (в данной версии не реализовано)
